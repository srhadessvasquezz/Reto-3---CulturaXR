## Extracto

**SivAR** es una plataforma de realidad aumentada para el patrimonio cultural de El Salvador, desarrollada como parte del **Reto 3: CulturaXR** (Turismo Creativo I). La plataforma permite a cualquier persona con un navegador y una cámara web explorar sitios arqueológicos, monumentos nacionales y piezas históricas mediante modelos 3D, detección de gestos con MediaPipe Hands e información histórica contextual.

Tecnologías: React 19, Vite 6, TypeScript 5, Three.js, React Three Fiber, MediaPipe Hands, MapLibre GL.

Colaboraron 5 contribuyentes en el repositorio compartido, con PRs por feature y merges contra `main`. El build de TypeScript y Vite pasa sin errores.
